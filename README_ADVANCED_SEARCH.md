# v1.2 Advanced Search

## รูปแบบการค้นหา

- `สำเริง วันเกิด`  
  ต้องมีทั้ง “สำเริง” และ “วันเกิด” ในไฟล์เดียวกัน โดยไม่ต้องอยู่ติดกันหรือเรียงลำดับกัน

- `สำเริง | สมชาย`  
  มี “สำเริง” หรือ “สมชาย” อย่างใดอย่างหนึ่ง

- `"นายสำเริง ใจดี"`  
  ค้นวลีตรงตัว

- `สำเริง -เชียงใหม่`  
  ต้องมี “สำเริง” แต่ตัดไฟล์ที่มี “เชียงใหม่”

- `สำเริ*`  
  Wildcard: ค้นข้อความที่มีรูปแบบใกล้เคียง/ต่อท้าย

## เพิ่มเติม

- รองรับช่องว่างหลายช่อง
- ทำ Unicode normalization สำหรับภาษาไทย รวมกรณี “ำ” ที่บันทึกคนละรูปแบบ
- ค้นพร้อมกันในชื่อไฟล์ โฟลเดอร์ และเนื้อหา
- แนะนำคำค้นระหว่างพิมพ์จากประวัติ ชื่อไฟล์ และโฟลเดอร์
- ฐานข้อมูล v1.0/v1.1 จะถูกอัปเกรดอัตโนมัติเมื่อแอปเริ่มทำงาน
- ไม่ต้องสแกนใหม่เพื่อให้ `สำเริง วันเกิด` ใช้งานได้ แต่ควรกดสแกนใหม่ทั้งหมดถ้าเพิ่งเพิ่ม OCR/.doc/.xls จาก v1.1
